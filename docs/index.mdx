组件时代的\
**Markdown**
=================

MDX 允许您在 Markdown 内容中使用 JSX。  
您可以导入组件（例如交互式图表或警告框）并将其嵌入内容中，  
这让编写包含组件的长文变得轻松愉快。 {/* more */}
🚀

## MDX 能做什么？

<div className="big-columns">
  <div>
    您可以通过 JSX 在 Markdown 中嵌入组件：

    ```mdx path="example.mdx"
    import {Chart} from './snowfall.js'
    export const year = 2023

    # 去年的降雪量

    {year} 年的降雪量高于平均水平。  
    随后温暖的春季导致许多附近河流出现洪涝状况。

    <Chart color="#fcb32c" year={year} />
    ```
  </div>

  <div>
    它会编译成 JavaScript，可在任何支持 JSX 的框架中使用：

## MDX 核心优势

<div className="emoji-list">
  * ❤️ **强大**：MDX 融合 Markdown 与 JSX 语法，完美契合基于 JSX 的项目  
  * 💻 **万物皆组件**：在 MDX 中使用现有组件，或将其他 MDX 文件作为组件导入  
  * 🔧 **可定制**：自由决定每个 Markdown 结构对应的渲染组件（`{h1: MyHeading}`）  
  * 📚 **基于 Markdown**：保留 Markdown 的简洁优雅，仅在需要时使用 JSX  
  * 🔥 **极速体验**：MDX 没有运行时，所有编译都在构建阶段完成  
</div>
